# 🧩 More Template

This folder contains random ready-to-use n8n workflows.  
Feel free to import and customize them for your own needs.

## 📁 File Format
All workflows are in `.json` format, exported directly from n8n.

## 🚀 How to Use
1. Open your n8n instance.
2. Go to the dashboard and click **Import from File**.
3. Select the `.json` file you want to use.
4. After import, review all nodes and replace any credentials with your own (e.g., database, API keys, email, etc.).
5. Adjust any node settings if needed, then activate or test the workflow.
